x=0
while(x<length)
 print "a(";x;"):",a(x):

---

It's actually quite easy to translate between C programs and FreeBasic programs because the manual contains a page detailing what the differences are.

https://www.freebasic.net/wiki/wikka.php?wakka=TblComparisonC

Of course it doesn't mention everything but it does cover important details such as the way variables are declared. Also a regular colon : is used to separate statements in FreeBasic instead of semicolon ; as in C. The other notable difference is that for, while, and if flow control structures are slightly different.
