## Pascal Hello World

```
program main;
begin
 writeLn('Hello World!');
end.
(*
 This is a Pascal program.
 It prints a hello message.

 compile and run as:

 fpc main.pas && ./main
*)
```

## C Hello World

```
#include <stdio.h>
int main()
{
 printf("Hello World!\n");
 return 0;
}
/*
 This is a C program.
 It prints a hello message.

 compile and run as:

 gcc main.c -o main && ./main
*/
```
